## Stack elegido para ESTE proyecto
- **Lenguaje:** PHP 8.2 (MVC propio de nsSkeleton)
- **Base de datos:** MySQL / MariaDB
- **CSS/JS:** Tailwind (compilado) + Alpine.js + Chart.js
- **IA / herramienta agéntica:** Claude Code (arranque por `AGENTS.md`)
- **Hosting / deploy:** hosting compartido con PHP 8.2; deploy por FTP desde el `.env`

## Por qué
Se usa el stack por defecto de nsSkeleton: liviano, sin dependencias pesadas, fácil de
deployar en hosting compartido, y ya trae login, cron, emails, backups, gráficos, file
manager, API y conector de IA listos. El equipo no necesita tooling de Node.
