Framework agéntico + sistema base

Empezá cualquier sistema de software en minutos, no en semanas

nsSkeleton es el esqueleto para iniciar proyectos de software de forma acelerada: una capa agéntica (agnóstica a la IA) que define cómo se desarrolla, y un sistema base ya programado con todo lo que cualquier sistema necesita.

⬇️ Descargar v1.9.0 Guía de instalación

PHP 8.2 · MySQL · Tailwind + Alpine · Sin dependencias pesadas

¿Para qué sirve?

Pensado para arrancar cualquier sistema (facturación, depósito/WMS, stock, turnos, restaurante… lo que sea) sin reinventar la base ni la metodología de desarrollo.

🤖

Capa agéntica (agnóstica a la IA)

Reglas, agentes, skills, comandos, plantillas y metodología (sprints, testing, bug-tracking, logging obligatorio). Markdown neutral: cualquier IA arranca leyendo AGENTS.md. Sin carpetas propietarias.

🧩

Sistema base ya programado

Un backend funcional con lo que todo sistema necesita: login, tareas programadas, emails, backups, gráficos, archivos y conector de IA. Sobre PHP con MVC propio + MySQL, listo para construir encima.

Qué trae el sistema base

🔐

Login + perfiles

Backend de administración con sesión segura, CSRF y gestión de administradores.

⏰

Cronmaster

Programador de tareas con expresiones cron, historial y ejecución manual.

✉️

Emails

Cliente SMTP propio (TLS/SSL), configuración por panel y log de envíos.

💾

Backups

Copias de base (SQL) y archivos (ZIP), restauración y retención automática.

📊

Gráficos + dashboard

Barras, torta y líneas con Chart.js, alimentados con datos del sistema.

📁

File manager

Carpetas/subcarpetas y subida de archivos, con protección anti-traversal.

🧠

Conector de IA

OpenAI y Deepseek: credenciales configurables y prueba de chat integrada.

🧱

Núcleo MVC

Router, PDO, vistas, auth y sesiones en PHP puro, sin dependencias.

Guía de instalación

  1. 1

    Descomprimir y configurar el entorno

    Copiá .env.example a .env y completá los datos de la base de datos (DB_*).

  2. 2

    Crear la base y el admin

    php system/database/migrate.php
php system/database/seed.php

    Admin inicial: admin@nsskeleton.local / admin1234.

  3. 3

    Levantar la aplicación

    php -S 127.0.0.1:8080 -t system/public system/public/index.php

    Entrá a /admin/login. (O apuntá Apache a system/public/.)
