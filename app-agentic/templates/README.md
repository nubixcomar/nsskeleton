# app-agentic/templates/ — Plantillas del proyecto

Plantillas propias del sistema u **overrides** de las del core
([`../../agentic/templates/`](../../agentic/templates/)). Una plantilla acá con el mismo
nombre que una del core tiene prioridad para tu proyecto.

Usos típicos: plantilla de doc de módulo adaptada a tu dominio, formato de reporte propio,
checklist de release específico del sistema.
