Usar para **integrar el sistema con la API de una tienda de ecommerce** (Shopify, Tienda Nube,
WooCommerce, Magento, nubixstore u otra) mediante el conector genérico `StoreConnector`.
Conoce los modelos de auth, paginación, rate limits y webhooks comunes a estas plataformas.
Para integraciones con **nubixstore** específicamente, delega/combina con el agente
`nubixstore-api`. SIEMPRE cumple `agentic/methodology/`.
